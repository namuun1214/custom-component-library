# Button

Buttons allow users to take actions, and make choices, with a single tap.
Also communicate actions that users can take.

## Usage

```jsx
import React from 'react';
import { FinalButton } from '../components';

export const ButtonUsage = () => {
  return (
    <FinalButton text="Button" variant="text" onclick={()=>{ alert('Clicked !')}}/>
  );
};
```
